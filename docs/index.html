Project Me vs You

This is a project I made at a recent hackathon in the category of games. I used the turtle library to create a game inspired by snake but it was a two player game. The game was called me vs you and the goal was to eat the food and grow your snake while trying to avoid colliding with the other snake. The game was a lot of fun to make and play and I learned a lot about how to use the turtle library to create games.

Python Turtle Time Random
More details

Problem

The problem was to create an engaging two-player snake game using the turtle library in Python.

Approach

I built the game step by step, starting with simple snake movement and then adding a second player. I kept the code organized by putting movement and reset behavior into separate functions like move_snake and reset_snake. I also used Git to track changes and tested the game carefully to make sure collisions and boundaries work correctly.

Results & Impact

The final result is a working two-player snake game made with Turtle. I improved the game by replacing hardcoded settings with variables so it is easier to adjust difficulty. The cleaner reset logic also fixed crashes when both snakes collided, making the game more stable and fun to play.

AWS

The goal of this project was to explore AWS by picking a stackholder question and then using different Aws services like Atena and S3 as well as sagemaker to answer the question.

sklearn Python AWS Sagemaker io
More details

Problem

Compare sagemaker to regular sklearn to figure out which machine learning model was more accurate in predicting how long it took for the city departments to resolve complaints.

Approach

The approach was to first use Athena to create a binary classification of the data set in order to predict whether a complaint would be resolved in less than 3 days or not. Then I used sagemaker to train a machine learning model on the data set and then I used sklearn to train a machine learning model on the same data set. Finally I compared the results of the two models to see which one was more accurate.

Results & Impact

The results showed that the sagemaker model was more accurate than the sklearn model in predicting how long it took for the city departments to resolve complaints. This insight will help inform future machine learning projects and guide the selection of appropriate tools for different use cases.
